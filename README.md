For the job board to work, follow the following steps:
<br/><br/>
(1) Download the jobs folder
(2) Open command line on your computer
(3) Using cd navigate to where the jobs folder is
(4) Type in cd jobs
(5) Type in npm start
<br/><br/>
The job board should work.
